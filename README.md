# Bicitaxi Admin

Sistema administrativo para la gestion de bicitaxis desarrollado con:

- Node.js
- Express
- MySQL
- Docker
- Arquitectura Cliente-Servidor

El proyecto permite administrar:

- Bicitaxis
- Conductores
- Viajes
- Rutas

---

# Tecnologias utilizadas

## Backend

- Node.js
- Express

## Base de datos

- MySQL 8

## Contenedores

- Docker
- Docker Compose

## Arquitectura

- MVC (Model View Controller)
